# 💰 ZP_automation_generation

Генератор умного табеля учёта рабочего времени в `.xlsx`.  
Автоматически создаёт таблицу на год с расчётом зарплаты, часов и премий при графике 5/2.

> 📌 Проект создан для личного использования, но легко адаптируется под любые нужды.

---

## ✨ Возможности

- 📅 Табель на 365 дней с автоматическими разрывами между месяцами
- 🎨 Чередование цветов (жёлтый/серый) для удобного чтения
- 🇷🇺 Дни недели на русском: `Понедельник`, `Вторник`...
- 💵 Расчёт зарплаты: `часы × ставка + премия (ручной ввод)`
- 📊 Итоги по месяцам и году: часы, дни, ЗП без премии, премия, итого
- ⏱ Поддержка дробных часов: `0.25` (15 мин), `0.5` (30 мин), `0.75`, `1.5` и т.д.
- 🔢 Все формулы в Excel — меняй оклад в ячейке `B1`, и всё пересчитается

## 🚀 Быстрый запуск (Windows PowerShell)

### Вариант 1: Через Poetry (рекомендуется, как в проекте)

```powershell
# 1. Клонируй репозиторий
git clone https://github.com/DiamondDim/ZP_automation_generation.git
cd ZP_automation_generation

# 2. Установи Poetry, если нет: https://python-poetry.org/docs/#installation

# 3. Исправь зависимости (в pyproject.toml сейчас пусто!)
# Открой pyproject.toml и замени блок dependencies на:
# dependencies = ["openpyxl>=3.1.0"]
# Затем выполни:
poetry install

# 4. Запусти в виртуальном окружении Poetry
poetry run python create_table.py
```

### Вариант 2: Через pip (проще, если не хочешь возиться с Poetry)

```powershell
# 1. Клонируй репозиторий
git clone https://github.com/DiamondDim/ZP_automation_generation.git
cd ZP_automation_generation

# 2. Установи зависимость
pip install openpyxl

# 3. Запусти скрипт
python create_table.py
```

### ✅ Результат

В папке проекта появится файл:
📄 Табель_учета_2026.xlsx
Открывай в Excel / LibreOffice / Google Таблицах.

### ⚙️ Настройка под себя

Открой create_table.py и найди блок НАСТРОЙКИ (строки ~10-15):
```
settings = [
    ("Месячный оклад:", 50000),      # ← Твой оклад
    ("Часов в рабочем дне:", 8),     # ← Норма часов
    ("Среднее рабочих дней в мес.:", 21.75),
    ...
]
```

Меняй значения → перезапускай скрипт → получай новый файл с твоими параметрами.
💡 Хочешь сгенерировать табель на другой год? Найди start_date = date(2026, 1, 1) и поменяй год.

### 🛠 Частые проблемы и решения

```
❌ ModuleNotFoundError: No module named 'openpyxl'
Выполните pip install openpyxl или исправь pyproject.toml и сделай poetry install
❌ Ошибка выполнения скриптов в PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser → введи Y
❌ Формулы в итогах показывают 0
Убедись, что в столбце Часы вводишь числа (0.5, 8), а не текст. Даты должны быть в формате ДД.ММ.ГГГГ
❌ Файл перезаписывается при каждом запуске
Перед запуском переименуй старый .xlsx или перенеси его в папку backup/
```
